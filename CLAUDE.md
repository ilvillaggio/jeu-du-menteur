# Règles permanentes — Il Villaggio

## Vocabulaire raccourci

Quand Fred dit **"bonjour"** → exécute immédiatement :
1. `git pull`
2. `git log -5 --oneline`
3. Résume en 2-3 lignes simples ce qui a été fait récemment

Quand Fred dit **"on a fini"** → exécute immédiatement :
1. Analyse les fichiers modifiés
2. Choisis un message de commit clair et court
3. `git add .` + `git commit` + `git push`
4. Confirme : "C'est envoyé sur GitHub ✅"

Quand Fred dit **"sauvegarde et balance"** → exécute immédiatement :
1. `git add .` + `git commit` + `git push`

Quand Fred dit **"mets-toi à jour"** → exécute immédiatement :
1. `git pull`

## Règle d'or
Si la conversation semble se terminer sans push, demande : "tu veux que je fasse 'on a fini' avant de quitter ?"
